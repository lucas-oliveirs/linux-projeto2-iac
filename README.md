# linux-projeto2-iac

Projeto simples de Infraestrutura como Código (IaC) em ambiente Linux.
---

## ⚙️ Comandos usados

- `apt update` → Atualiza a lista de pacotes
- `apt upgrade -y` → Atualiza os pacotes instalados
- `mkdir` → Cria diretórios
- `chmod` → Altera permissões
- `chown` → Altera dono e grupo
- `echo` → Escreve texto em arquivos
- `cat` → Exibe conteúdo de arquivos
- `mv` → Move ou renomeia arquivos
- `rm` → Remove arquivos ou diretórios
